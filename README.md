# 🎮 Resurge - Call of Duty Clients

https://resurge.gamer.gd

Resurge é um projeto educacional que demonstra o desenvolvimento de clients modernos para jogos clássicos da série Call of Duty, permitindo que os usuários joguem em servidores Xbox Live em vez dos tradicionais servidores Steam.

Este projeto é fornecido apenas para fins educacionais e de pesquisa.

## 📚 Sobre o Projeto

Resurge é um projeto educacional focado em:

- **Reverse Engineering**: Compreensão de protocolos de rede e estruturas de cliente de jogos
- **Desenvolvimento de Clientes**: Criação de aplicações cliente robustas e seguras
- **Segurança**: Implementação de práticas seguras de comunicação entre cliente e servidor
- **Qualidade de Vida**: Melhorias modernas em clientes legados

## 🛠️ Tecnologias Utilizadas nos Clients e no Website

### Frontend
- **React** - Interface de usuário moderna e responsiva
- **Electron** - Framework para aplicações desktop multiplataforma
- **Tailwind CSS** - Estilização e design responsivo
- **HTML/CSS/JavaScript** - Padrões web modernos

### Backend & Clientes
- **C++** - Desenvolvimento de clientes de alto desempenho
- **DirectX** - Renderização gráfica avançada
- **Xbox Live SDK** - Integração com servidores Xbox

### Deploy & DevOps
- **GitHub Pages** - Hospedagem do website
- **GitHub Actions** - CI/CD automation
- **Google AdSense** - Monetização do website

## 🎯 Clientes Disponíveis

### Resurge Black Ops III (BETA)
- ✅ Jogue Zombies em servidores Xbox
- ✅ Seguro contra exploits Steam
- ✅ Patches de segurança mais recentes
- ✅ Melhor performance em sistemas modernos
- ✅ Melhorias de controle
- ✅ Suporte a mods (versão MS Store)

### Resurge Black Ops II
- ✅ Experiência vanilla em servidores Xbox
- ✅ Upgrade DirectX 11 64-bits (vs DX9 32-bits)
- ✅ Proteção contra exploits de rede Steam
- ✅ Suporte completo a Zombies e Multiplayer

### Legacy Clients
- **Black Ops I** - Clássico multiplayer e Zombies
- **Modern Warfare 2** - Experiência fast-paced original
- **Infinite Warfare / MWR** - Futurista e remastered

## 📖 Como Funciona

### Para Black Ops III
1. Download do instalador BETA via GitHub
2. Instalação em qualquer diretório (fora da pasta do jogo)
3. Abertura do launcher
4. Seleção da pasta do jogo
5. Instalação do Resurge
6. Lançamento via launcher ou Resurge.exe

### Para Black Ops II e Legacy
- Entre em contato via [Discord](https://discord.gg/tQxwsCRkQU)
- Abra um ticket
- Receba o cliente pronto para jogar
